# Quản lý Cửa hàng - Project SBA301

Dự án sba301

mục tiêu: có thể hoàn thành lấy kinh nghiệm để từ đấy làm project cá nhân hoàn chỉnh 
bằng spring + reacts

## 📁 Cấu trúc thư mục
- **/backend**: Mã nguồn Spring Boot (Java), xử lý API.
- **/frontend**: Mã nguồn ReactJS (JavaScript), giao diện người dùng.
- **/docs**: File script SQL tạo cơ sở dữ liệu (`sba301.sql`).

## 🛠 Công nghệ sử dụng
- **Backend**: Spring Boot 3, Spring Data JPA, SQL Server.
- **Frontend**: ReactJS, Axios, Bootstrap 5.

## 🚀 Hướng dẫn chạy nhanh
1. **Backend**: Chạy `ManagementApplication.java` tại port 8080.
2. **Frontend**: Vào thư mục `frontend`, chạy `npm start` tại port 3000.
